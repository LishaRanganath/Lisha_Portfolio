# Lisha Ranganath — Portfolio

Personal portfolio website built with React. Showcases projects, publications, certifications, professional experience, and education.

🔗 **Live:** [http://localhost:3000](http://localhost:3000)

---

## Sections

- **Introduction** — Intro with bio snapshot, skill tags, and resume download
- **Certificates** — Course certifications with descriptions
- **Projects** — Full-stack, distributed systems, and ML projects with highlights and tech tags
- **Publications** — IEEE published research paper
- **Education** — Academic background (Georgia Tech MSCS + NHCE B.E.)
- **Experience** — Professional work history (L&T Technology Services, Plan Source India)
- **Contact** — Phone, email, and address
- **Footer** — LinkedIn and GitHub links

---

## Tech Stack

- **React** (Create React App)
- **CSS Modules** (per-component stylesheets)
- **Google Fonts** — Montserrat
- **Icons** — Icons8 (via CDN)

---

## Project Structure

```
src/
├── assets/               # Images
├── components/
│   ├── Header/           # Fixed nav with hamburger menu
│   ├── Introduction/     # Introduction section — animated intro + bio card
│   ├── Certificates/     # Certificate cards grid
│   ├── Projects/         # Project cards with highlights and tags
│   ├── Publications/     # IEEE publication with link
│   ├── Education/        # Education timeline
│   ├── About/            # Professional experience timeline
│   ├── Contact/          # Contact info cards
│   └── Footer/           # Social links
├── App.js                # Root component
└── index.css             # Global styles + keyframe animations
```

---

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```
